Linux常用的解压缩命令

tar.gz linux自带 [code] 解压: tar -zxvf FileName.tar.gz //解压FileName.tar.gz文件 压缩：tar -zcvf DirName.tar.gz DirName //将文件夹DirdName压缩为 DirName.tar.gz tar -zcvf FileName.tar.gz FileName1 FileName2 … //将FileName1,FileName2…压缩为FileName.tar.gz [/code]

.zip 需要安装zip以及unzip [code] 解压：unzip FileName.zip //解压FileName.zip 压缩：zip -r DirName.zip DirName //将文件夹DirName压缩为DirName.tar.gz [/code]

.rar 需要下载rar [code] 解压：rar x FileName.rar //解压FileName.rar 压缩：rar a FileName.rar DirName //将DirName压缩为DirName.rar [/code]

.7z 需要下载p7zip-full [code] 解压：7z x FileName.7z //解压FileName.7z 压缩：7z a DirName.7z DirName.7z //将DirName压缩为DirName.7z [/code]

安装命令示例： Redhat、Fedora、Centos下安装zip、unzip： [bash] yum install -y zip unzip [/bash] Debian、Ubuntu下安装zip、unzip： [bash] apt-get install -y zip unzip [/bash]
