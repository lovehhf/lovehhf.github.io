ubuntu 14.04服务器上使用nginx搭建wordpress博客

过程详解

1. 更新apt-get

[bash] sudo apt-get update [/bash]

2.安装nginx

[bash] sudo apt-get install nginx [/bash]

3.启动nginx [bash] sudo service nginx start [/bash]

此时浏览器输入你的服务器ip会看到如下界面

4. 安装mysql

[bash] sudo apt-get install mysql-server php5-mysql [/bash]

5. 安装php

[bash] sudo apt-get install php5-fpm [/bash]

6. 配置PHP

[bash] sudo vi /etc/php5/fpm/php.ini [/bash] 找到cgi.fix_pathinfo=1,把1改成0,并去掉；号(如果vim编辑器的上下左右键变成了ABCD可以使用下面的命令重装vim解决) [bash] sudo apt-get remove vim-common sudo apt-get install vim [/bash]

7. 启动php-fpm

[bash] sudo service php5-fpm restart [/bash]

8. 配置nginx

[bash] sudo vi /etc/nginx/sites-available/default [/bash] 打开后如图： 改为 取消一些注释如图： 保存退出并重启nginx [bash] sudo service nginx restart [/bash]

9. 安装phpmyadmin,并建立软链接

[bash] sudo apt-get install phpmyadmin sudo ln -s /usr/share/phpmyadmin/ /usr/share/nginx/html/phpmyadmin [/bash] 安装中会要求输入密码 浏览器访问phpmyadmin:ip/phpmyadmin 登录账号为root，密码为设置的密码

10 建立名为wordpress的数据库

(如果你擅长SQL语句，可以直接用SQL语句创建数据库 create database wordpress;)

11. 下载wordpress,解压，转移到nginx目录,更改文件所有者

[bash] cd /usr/share/nginx/html/ #有的是www而不是html sudo wget https://cn.wordpress.org/wordpress-4.3.1-zh_CN.tar.gz sudo tar -zxvf wordpress-.tar.gz sudo mv wordpress/ . sudo rm index.html sudo chown -R www-data:www-data * sudo service nginx restart [/bash]

12. 此时浏览器输入ip访问会出现如下页面

点击现在就开始,填表,提交。

如果提示抱歉，但是向导在您的文件系统中没有足够的权限写入wp-config.php文件。,使用下面的命令可直接修改写入wp-config.php

[bash] sudo cp wp-config-sample.php wp-config.php sudo vi wp-config.php [/bash]

填写站点信息完成安装

13. 此时nginx更改固定链接会提示404,在nginx配置文件加入rewrite规则可解决.

解决：

[bash] sudo vi /etc/nginx/sites-available/default 写入 if (-f $request_filename/index.html){ rewrite (.) $1/index.html break; } if (-f $request_filename/index.php){ rewrite (.) $1/index.php; } if (!-f $request_filename){ rewrite (.*) /index.php; } [/bash]

此时固定链接就正常了。

至此wordpress安装完成。
